# Glance

Building a scheduling app with [Gleam](https://gleam.run/) and [Lustre](https://hexdocs.pm/lustre/index.html)

[View on GH pages](https://stewartrule.github.io/Glance/)

## Development

```sh
gleam run -m lustre/dev start
```

## Build

```sh
gleam run -m lustre/dev build app --minify
```
